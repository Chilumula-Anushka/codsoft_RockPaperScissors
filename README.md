# codsoft_RockPaperScissors
A simple command-line **Rock Paper Scissors** game developed using **Python** as part of the **CodSoft Python Programming Internship**. The game allows the user to play against the computer with randomly generated choices.

## 📌 Features
- Interactive command-line interface
- Random computer choice
- User-friendly input
- Win, Lose, or Draw result
- Play multiple rounds
- Input validation for invalid choices
